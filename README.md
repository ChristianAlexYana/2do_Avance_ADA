# Buscador de Patrones - Algoritmos de Búsqueda

Aplicación C++ que implementa y compara diferentes algoritmos de búsqueda de patrones en texto.

## Características

- **Algoritmos implementados:**
  - KMP (Knuth-Morris-Pratt)
  - Boyer-Moore
  - Rabin-Karp
  - Búsqueda Naive (Fuerza Bruta)

- **Funcionalidades:**
  - Búsqueda de patrones en texto
  - Carga de archivos TXT
  - Análisis de secuencias de ADN
  - Medición de tiempo de ejecución
